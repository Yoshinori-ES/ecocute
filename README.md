# 地域のエコキュート専門店 - サービスサイト

## プロジェクト概要
- **名称**: 地域のエコキュート専門店（株式会社イーエス・テック）
- **目的**: BtoC事業立ち上げ用のLP型サービスサイト
- **ターゲット**: 静岡県袋井市および周辺エリアにお住まいの一般家庭
- **技術スタック**: Hono + TypeScript + Tailwind CSS (CDN) + Cloudflare Pages

## サイト構成（全7ページ + API）

| パス | ページ名 | 内容 |
|------|---------|------|
| `/` | トップページ | ファーストビュー・5つの強み・FAQ・CTA |
| `/leak` | 漏水・故障ガイド | 緊急対応4ステップ・漏水種類別対処法 |
| `/error-codes` | エラーコード一覧 | 7メーカー網羅（タブ切替・色分け判定） |
| `/company` | 会社概要・資格 | 許認可5件・資格18種・Googleマップ |
| `/flow` | ご依頼の流れ | 9ステップのフロー図 |
| `/area` | 対応エリア | 10市町一覧・Googleマップ |
| `/subsidy` | 補助金（給湯省エネ2026） | 補助額・条件・申請フロー |
| `/api/health` | ヘルスチェック | JSON形式のステータス |

## 実装済み機能

### コア機能
- スマホファーストのレスポンシブデザイン
- フローティングCTA（電話・メールボタン、スマホ下部固定）
- 対話型AI電話受付の訴求
- FAQ アコーディオン
- セクションフェードインアニメーション
- エラーコード7メーカー タブ切替表示
  - Panasonic / 三菱電機 / 日立 / 東芝 / ダイキン / コロナ / ノーリツ
  - 緑（自己解決可）/ 黄（要確認）/ 赤（修理必要）の3段階判定

### SEO/GEO対策
- 全ページにtitle / description / canonical / OGP設定
- LocalBusiness 構造化データ（JSON-LD）
- HowTo 構造化データ（漏水ページ）
- 地域キーワード最適化（袋井市・磐田市・掛川市等）
- パンくずリスト
- Googleマップ埋め込み（会社概要・エリアページ）

### DX独自機能
- 対話型AI電話受付の訴求（全ページCTA）
- 050番号によるIP電話対応
