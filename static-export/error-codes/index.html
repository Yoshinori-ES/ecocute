エラーコードのリセット方法（共通）

  1. 1リモコンのエラーコードをメモしてください
  2. 2エコキュート本体の漏電ブレーカーをOFFにします（1分以上待つ）
  3. 3再度ブレーカーをONにし、リモコンの表示を確認
  4. 4エラーが再表示される場合は修理が必要です → 当店にご連絡ください

エラーが解消しない場合は
お気軽にご連絡ください
